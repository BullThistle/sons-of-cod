## Building the App or Contributing

* Clone this repository
* Install required dependencies

```
$ bundle install
```

* To run local server

```
$ rails s
```

* To set first user as admin
```
$ rails c
$ @user = User.first
$ @user.admin = true
$ @user.save
$ exit
```
